APPL
Associative Pseudoword Learning — Interactive Demo
About this demo
In this demo you will practice learning novel pseudowords paired with object images — a short version of the full APPL paradigm. You will see an image and a word below it and decide whether that word is the correct name for the image or not. Feedback is given after every trial so you can learn the associations.
Controls
Use keyboard keys or click the on-screen buttons during the task.
1 This IS the correct word (Ja / Yes)
2 This is NOT the correct word (Nein / No)

How it works
1
An image appears on a dark background with a word displayed below it.
2
Decide: is this word the correct name for the image? Press 1 for Yes, 2 for No. You have 2.5 seconds to respond.
3
You will always see feedback: "Korrekt, das ist X" or "Falsch, das ist X" — this is how you learn the correct associations.
4
The demo runs 2 short learning stages (10 trials each) with 5 pseudoword–image pairs. Try to memorise the correct names!
Tip
Focus on the image and say the word silently to yourself. The feedback always tells you the correct name — use it to build the memory trace.
