OLMM
Object Memory Location Mapping — Interactive Demo
About this demo
In this demo you will practise learning where objects are located on a spatial map. Each trial shows a house placed at a specific position on a map. You decide whether this position is the correct location for that house or a false location. Feedback after every trial helps you build the memory trace.
Controls
Use keyboard keys or click the on-screen buttons.
← Left Correct position for this house
Right → Incorrect / false position
1 / 2 / 3 Select option in the final recognition test
← Back to Paradigm Overview
How it works
1
A map image appears on a dark background showing a house placed at a particular location.
2
Decide: is this the correct location for that house? Press ← Left for Yes (correct) or Right → for No (false position). You have 2.5 seconds.
3
Feedback always shows the correct map position — use it to memorise where each house belongs.
4
After 2 learning repetitions you will complete a short 3-alternative forced-choice recognition test (keys 1, 2, 3) for each house.
Tip
The map images are encoded in the filename: hH_pP_k_direction_side = correct location; _i_ = false position. Try to form a vivid spatial memory for each house's true position!
